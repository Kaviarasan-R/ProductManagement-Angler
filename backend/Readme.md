# Backend Using Node JS

Create .env file and add below credentials to it.

PORT=5000

JWT_SECRET=<?>

MONGO_URI=<?>

CLOUDINARY_CLOUD_NAME=<?>

CLOUDINARY_KEY=<?>

CLOUDINARY_SECRET=<?>

MAIL_ID=<?>

MAIL_PASS=<?>


## We're using totally 8 api's.

1. Register
2. Login
3. Forgot Password
4. Add Product
5. Update Product
6. Delete Product
7. Get Product
8. Search Product
